# models.dev-website

A faster website for browsing [models.dev](https://models.dev) data.

I created this fork because I spend a lot of time on models.dev every day and it bothers me that it's slow when searching for models and providers — so I rebuilt it to make the search faster.

Built with React + Vite.

## Quick start

```bash
bun install
bun run dev
```
